Cascading Syyle Sheets atau CSS adalah bahasa pemrograman berbasis arutan yang dikembangan oleh World Wide Web Consortium (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsu untuk memformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut. Misalnya, pada HTML versi 3.2, yang sudah diperkenalkan tag agar gaya text bisa diubah. namun web designer amatir menyalahgunakan dan tidak ada opsi fallback seperti yang sekarang tersedia di CSS. Jadi, pilihan terbaiknya adalah tetap memisahkan pemformatan (HTML) dan penataan gaya (CSS)
